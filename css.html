Some Basic CSS Information:

  • CSS stands for Cascading Style Sheets.
  • It is used for styling web pages written in HTML or XML.
  • CSS allows you to control the layout and appearance of your website.
  • Selectors are used to target HTML elements for styling.
  • Properties define how the selected elements should be styled.
